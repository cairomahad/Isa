# 📚 Инструкция для админа: Как добавлять уроки

## 🎯 Доступ к управлению уроками

### 1. Войдите как админ
```
Телефон: admin
Пароль: admin123
```

### 2. Откройте админ панель
- Нажмите на таб **"Ещё"** (Settings)
- Если вы админ, автоматически откроется админ панель

### 3. Перейдите к управлению уроками
- Нажмите **"Видео уроки"** → "Добавить / Редактировать уроки"
- Откроется экран со списком всех уроков

---

## ➕ Добавление нового урока

### Шаг 1: Нажмите кнопку "+"
В правом верхнем углу экрана нажмите круглую кнопку с плюсом

### Шаг 2: Заполните форму

**Обязательные поля:**

**1. Название урока** *
```
Например: Условия намаза
Например: Урок 1. Введение в фикх
```

**2. Категория** *
Выберите одну из категорий:
- 📘 **Шафиитский мазхаб** (fard)
- 📗 **Ханафитский мазхаб** (hanafi)
- 🔤 **Арабский язык** (arabic)
- 🏠 **Семейные отношения** (family)

**Опциональные поля:**

**3. Описание**
```
Краткое описание урока
Например: В этом уроке мы разберем основные условия намаза
```

**4. Видео File ID**
```
Telegram file_id или прямая ссылка на видео
Например: BAACAgIAAxkBAAIDzGlOxyVtoVpYbqUHLf40Al8FmTUu
Или: https://example.com/video.mp4
```

**5. Аудио File ID**
```
Telegram file_id для аудиоверсии урока
Например: CQACAgIAAxkBAAID02lOz2VtoVpY...
```

**6. PDF File ID**
```
Telegram file_id для PDF материалов
Например: BQACAgIAAxkBAAID1GlO0GVtoVpY...
```

### Шаг 3: Нажмите "Создать урок"
Урок будет добавлен в базу данных и появится в списке

---

## ✏️ Редактирование урока

### 1. Найдите урок в списке
Все уроки отображаются карточками

### 2. Нажмите "Редактировать"
Откроется та же форма, но с заполненными полями

### 3. Измените нужные поля
Можно изменить:
- Название
- Описание
- Категорию
- Файлы (видео, аудио, PDF)

### 4. Нажмите "Сохранить изменения"

---

## 🗑️ Удаление урока

### 1. Нажмите кнопку "Удалить" на карточке урока

### 2. Подтвердите удаление
Появится диалог подтверждения

⚠️ **Внимание:** Удаление нельзя отменить!

## 📂 Как получить File ID из Telegram?

### Вариант 1: Через Telegram Bot

1. Создайте бота через @BotFather
2. Отправьте файл (видео/аудио/PDF) вашему боту
3. Получите file_id через Telegram Bot API:
```python
import telegram
bot = telegram.Bot(token='YOUR_BOT_TOKEN')
updates = bot.get_updates()
# Найдите file_id в updates
```

### Вариант 2: Использование прямых ссылок

Если файлы хранятся в облаке (Google Drive, Dropbox, etc):
1. Получите **прямую ссылку** на файл
2. Вставьте ссылку в поле File ID
3. Убедитесь, что ссылка публичная и доступна

### Вариант 3: Загрузка в Supabase Storage (рекомендуется)

1. Откройте [Supabase Dashboard](https://supabase.com/dashboard)
2. Перейдите в **Storage**
3. Создайте bucket (например, `lessons`)
4. Загрузите файлы
5. Скопируйте публичный URL
6. Используйте этот URL в качестве File ID

## 📊 Структура категорий

### fard (Шафиитский мазхаб)
- Основы исламского права по шафиитскому мазхабу
- Обязательные знания для мусульман

### hanafi (Ханафитский мазхаб)
- Основы исламского права по ханафитскому мазхабу
- Альтернатива шафиитскому мазхабу

### arabic (Арабский язык)
- Уроки арабского языка
- Открываются после завершения базовых курсов

### family (Семейные отношения)
- Ислам в семейной жизни
- Открываются после завершения базовых курсов

## 🔄 Порядок открытия уроков для студентов

### Логика unlock:

1. **Первый урок** каждого курса всегда открыт
2. **Следующий урок** открывается через 3 дня после завершения предыдущего
3. **Арабский и Семейные отношения** открываются только после завершения Обязательных знаний (shafi или hanafi)

### Пример:
```
День 1: Студент проходит Урок 1
День 4: Открывается Урок 2 (через 3 дня)
День 7: Студент проходит Урок 2
День 10: Открывается Урок 3 (через 3 дня)
```

## 💡 Советы и рекомендации

### 1. Названия уроков
✅ **Хорошо:**
- "Урок 1. Введение в фикх"
- "Условия намаза"
- "Столпы Ислама"

❌ **Плохо:**
- "урок"
- "1"
- "смотреть"

### 2. Описания
✅ **Хорошо:**
- Краткое и информативное
- 1-2 предложения
- Без форматирования

❌ **Плохо:**
- Слишком длинное описание
- Много абзацев
- HTML теги

### 3. Порядок добавления
1. Сначала добавьте все уроки для **Шафиитского мазхаба**
2. Затем для **Ханафитского мазхаба**
3. Потом **Арабский язык**
4. И наконец **Семейные отношения**

### 4. Файлы
- **Видео:** Рекомендуется MP4, не более 100 MB
- **Аудио:** MP3, не более 50 MB
- **PDF:** Не более 20 MB

## 🔧 Технические детали (для разработчиков)

### Backend API endpoints:

```
GET    /api/admin/lessons          - Список всех уроков
POST   /api/admin/lessons          - Создать урок
PUT    /api/admin/lessons/{id}     - Обновить урок
DELETE /api/admin/lessons/{id}     - Удалить урок
```

### Структура данных:

```json
{
  "title": "Название урока",
  "description": "Описание",
  "category": "fard",
  "video_file_id": "telegram_file_id_or_url",
  "audio_file_id": "telegram_file_id_or_url",
  "pdf_file_id": "telegram_file_id_or_url"
}
```

### Таблица в Supabase:
- **Таблица:** `video_lessons`
- **Поля:** id, title, description, category, file_id, audio_file_id, pdf_file_id, created_at

## ❓ FAQ

**Q: Можно ли загружать видео напрямую в приложение?**  
A: Сейчас нет, нужно использовать file_id из Telegram или прямые ссылки. В будущем добавим прямую загрузку.

**Q: Как изменить порядок уроков?**  
A: Порядок определяется автоматически по ID урока (время создания). Сначала удалите и создайте заново в нужном порядке.

**Q: Можно ли добавить домашнее задание к уроку?**  
A: Да, домашние задания добавляются отдельно через раздел "Домашние задания" (функция в разработке).

**Q: Сколько уроков можно добавить?**  
A: Ограничений нет, можно добавить сколько угодно уроков.

**Q: Как студенты увидят новые уроки?**  
A: Сразу после добавления, уроки появятся в разделе "Уроки" у всех студентов.

# Инструкция по сборке APK — Tazakkur

---

## Шаг 1 — Задеплоить backend на Railway

Перед сборкой APK нужно обновить backend (файл `server.py` был изменён).

**Вариант A — через Railway Dashboard:**
1. Открой https://railway.app → твой проект
2. Зайди в Variables → убедись что есть `SUPABASE_URL` и `SUPABASE_ANON_KEY`
3. В разделе Deployments → Redeploy последний деплой
   (или пушни изменения в GitHub если Railway подключён к репозиторию)

**Вариант B — через GitHub (если репозиторий подключён к Railway):**
```bash
# Из корня проекта
git add backend/server.py
git commit -m "feat: YouTube, quiz fix, leaderboard, push notifications"
git push origin main
```
Railway автоматически пересоберёт.

## Шаг 2 — Сборка APK

### Требования
- Аккаунт на https://expo.dev (бесплатно)
- Node.js 18+ установлен
- EAS CLI: `npm install -g eas-cli`

### Команды

```bash
# 1. Войти в Expo
eas login

# 2. Перейти в папку с приложением
cd /путь/к/проекту/frontend

# 3. Запустить сборку APK
eas build --platform android --profile preview
```

Сборка будет идти **5–15 минут** на серверах Expo.  
По завершении — ссылка на скачивание APK в консоли и на expo.dev.

> **eas.json уже настроен** с правильным `EXPO_PUBLIC_BACKEND_URL = https://tazakkur-production-c8c9.up.railway.app`

## Что проверить в APK

### Кеширование аккаунта (основной фикс)
1. Войди в аккаунт
2. Полностью закрой приложение (свайп в Recent Apps)
3. Открой снова → **должен остаться залогиненным**

### YouTube видео (только в APK — не работает в Expo Go)
- Уроки → Шафиитский мазхаб → урок "Тестовый урок"
- Должен открыться YouTube-плеер прямо в приложении

### Кнопки громкости для Зикра (только в APK)
- Открой Зикр → нажми Volume Up / Volume Down
- Счётчик зикра должен увеличиваться

### Push-уведомления (только в APK)
- При первом запуске разрешить уведомления
- После завершения урока → через 3 дня придёт уведомление

### Квиз с вариантами ответов
- Открой урок → "Пройти тест"
- Варианты А/Б/В/Г отображаются
- Правильный → **зелёный** ✓, неправильный → **красный** ✗

### 5 вкладок в нижнем меню
```
Главная | Уроки | Зикр | Рейтинг | Ещё
```

## Частые проблемы

| Ошибка | Решение |
|--------|---------|
| `"This module is not supported in Expo Go"` | Так и должно быть — нужен APK |
| `Network request failed` | Backend не задеплоен на Railway, см. Шаг 1 |
| Выходит из аккаунта при закрытии | Исправлено — нужна пересборка APK |
| `Build failed: SDK version mismatch` | expo в package.json = `"54.0.33"` — должно совпадать |
| Пустой экран вместо видео | YouTube плеер не работает в Expo Go, только APK |

## Быстрое обновление без пересборки (OTA)

Если изменился только JavaScript-код (не нативные модули):

```bash
eas update --branch preview --message "fix: описание изменений"
```

Пользователи получат обновление автоматически при следующем открытии приложения.

> ⚠️ OTA НЕ работает для: новых npm пакетов с нативным кодом, изменений в `app.json`, новых Expo плагинов.

## Обновление версии APK

Перед каждой новой сборкой увеличь версию в `app.json`:
```json
{
  "expo": {
    "version": "1.0.1",
    "android": {
      "versionCode": 2
    }
  }
}
```
